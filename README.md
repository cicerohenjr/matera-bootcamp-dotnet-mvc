# 🚀 Bootcamp Desenvolvimento .NET Core MVC - Instituto Matera

Repositório dedicado aos projetos e anotações desenvolvidos durante o **Bootcamp de Desenvolvimento .NET Core MVC para Iniciantes** promovido pelo Instituto Matera.

## 🎯 Objetivo
Compreender a construção de software utilizando o ecossistema Microsoft. O objetivo desta imersão foi consolidar os fundamentos da linguagem **C#** e do framework **ASP.NET Core MVC**, complementando minha formação em Gestão de TI e Cloud Computing com a visão completa do ciclo de vida das aplicações.

## 🛠️ Tecnologias e Ferramentas Utilizadas
* **Linguagem:** C#
* **Framework:** .NET Core MVC
* **IDE:** Visual Studio Community 2022
* **Banco de Dados:** SQL Server & SSMS
* **ORM:** Entity Framework Core (Code First)

## 📂 Estrutura do Aprendizado (Trilha do Curso)

* **Dia 01: Fundamentos e Arquitetura**
  * Relação Cliente x Servidor e protocolo HTTP.
  * Padrão de arquitetura MVC (Model-View-Controller).
  * Configuração do ambiente no Visual Studio.

* **Dia 02: Integração e Banco de Dados (Hands-on)**
  * Modelagem de dados com SQL Server.
  * Acesso a dados com Entity Framework Core.
  * Criação de Views com Razor (HTML + C#) e Bootstrap.
  * Fluxo CRUD completo (Create, Read, Update, Delete).

* **Dia 03: Segurança e Deploy**
  * Autenticação e Autorização.
  * Testes funcionais e ajustes visuais.
  * Preparação para o ambiente de produção.
